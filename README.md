1.addVehicle
2.updateVehicle
3.deleteVehicle
4.scheduleTrip
5.cancelTrip
6.bookTrip
7.cancelBooking
8.allocateDriver
9.deallocateDriver
10.getBookingsByPassenger
11.getBookingsByTrip
12.getAvailableDrivers

There are 2 custom exceptions 
1.VechileNotFoundException
2.BookingNotFoundException
